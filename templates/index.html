แบบประเมินความเสี่ยงต่อการเป็นโรคหัวใจ

การแปลผลจากคะแนน คะแนน = 0 : พบว่า คุณไม่มีความเสี่ยง “โรคเส้นเลือดหัวใจ”
คะแนนมากกว่า 2 ขึ้นไป : พบว่า มีความเสี่ยงโรคเส้นเลือดหัวใจ และเกณฑ์ความเสี่ยงจะเพิ่มสูงขึ้นตามจำนวนคะแนน ที่เพิ่มขึ้นของคุณ แนะนำให้ปรึกษาแพทย์ผู้เชี่ยวชาญ

AgeGategory : กลุ่มอายุ
ก่อนหน้านี้ ต่อไป
1. BML: ค่าดัชนีมวลการ
2. PhysicalHealth : ระดับสุขภาพทางกาย
3. MentalHealth : ระดับสุขภาพทางจิต
4. SleepTime : เวลาในการนอน
ก่อนหน้านี้ ต่อไป
รายการ ใช่ ไม่ใช่
5. smoking : เคยสูบบุหรี่หรือไม่
6. AlcoholDrinking : ดื่มแอลกอฮอล์หรือไม่
7. Stroke : เป็นโรคหลอดเลือดสอมงหรือไม่
8. DiffWalking : มีปัญหาในการเดินหรือไม่
9. Asthma : เป็นโรคหอบหืดหรือไม่
10. KidneyDisease : เป็นโรคไตหรือไม่
11. SkinCancer ; เป็นโรคมะเร็งผิวหนังหรือไม่
13. Diabetic : เป็นโรคเบาหวานหรือไม่
14. PhysicalActivity : ออกกำลังกายใน 30 วันที่ผ่านมาหรือไม่

รายการ มากที่สุด มาก ปานกลาง น้อย น้อยที่สุด
